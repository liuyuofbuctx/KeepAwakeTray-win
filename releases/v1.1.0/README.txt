KeepAwakeTray v1.1.0
=====================

Version: v1.1.0

Changes
-------
- Added dual tray icons for runtime state.
- Active/effective keep-awake state uses the dark icon.
- Paused or error state uses the light icon.
- Default idle threshold remains 8 minutes.

Install
-------
1. Download KeepAwakeTray-Windows.zip.
2. Extract the zip file.
3. Run KeepAwakeTray.exe directly, or run Install-KeepAwakeTray.ps1 to install for the current Windows user and create a desktop shortcut.

Notes
-----
- The app is unsigned, so Windows SmartScreen may warn on first run.
- Right-click the tray icon to pause, resume, trigger once, or exit.
